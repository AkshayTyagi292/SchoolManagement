        private void btnLogin_Click(object sender, EventArgs e)
        {
            SqlConnection con = new SqlConnection(@"Data Source = ASUS\SQLEXPRESS; Initial Catalog = schooldb; Integrated Security = True; Encrypt = False");
            con.Open();
            string username = txtUsername.Text;
            string password = txtPassword.Text;
            SqlCommand cnn = new SqlCommand("select username, password from login where username='" + txtUsername.Text + "' and password='" + txtPassword.Text + "'",con);
            SqlDataAdapter da = new SqlDataAdapter(cnn);
            DataTable dt = new DataTable();
            da.Fill(dt);
            if(dt.Rows.Count > 0 )
            {
                Main mn = new Main();
                mn.Show();
            }
            else
            {
                MessageBox.Show("Invalid Username or Password");
            }
            con.Close();
        }
    }
}


Explanation:
Namespaces and Class Definition:

using System.Data;: Provides access to the ADO.NET classes like DataTable.
using System.Windows.Forms;: Provides access to Windows Forms classes.
using Microsoft.Data.SqlClient;: Provides access to SQL Server data provider classes.
using static System.Runtime.InteropServices.JavaScript.JSType;: Imports the JSType enum from JavaScript interop.
Form1 Class:

public partial class Form1 : Form: Defines a partial class Form1 that inherits from Form, which is a base class for all Windows Forms in C#.
Constructor (Form1()):

InitializeComponent();: Initializes the components of the form (typically generated by the designer).
Event Handlers:

label3_Click: Event handler for when label3 is clicked (currently empty).
Form1_Load: Event handler for when Form1 is loaded (currently empty).
btnLogin_Click: Event handler for when the login button (btnLogin) is clicked.
Login Button Click (btnLogin_Click):

Database Connection (SqlConnection con):

Establishes a connection to a SQL Server database (schooldb) on ASUS\SQLEXPRESS with integrated security (Integrated Security=True).
SQL Query (SqlCommand cmd):

Constructs a SQL command to select username and password from the login table where the username and password match those entered in the text boxes (txtUsername and txtPassword).
Data Retrieval (SqlDataAdapter da):

Uses SqlDataAdapter to execute the SQL command (cmd) and fill the results into a DataTable (dt).
Validation:

Checks if any rows were returned (if (dt.Rows.Count > 0)), indicating valid login credentials.
If valid, opens a new instance of Main form (Main mn = new Main(); mn.Show();).
If invalid, displays an error message using MessageBox.Show("Invalid Username or Password");.
Closing Connection:

Closes the database connection (con.Close();) after executing the query.
